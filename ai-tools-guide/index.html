🚀 Getting Started (5 Minutes)

Step 1: Install Ollama — one command, works on Mac/PC/Linux
$ curl -fsSL https://ollama.com/install.sh | sh
Step 2: Pull a model (Llama 3.2 is great for start)
$ ollama pull llama3.2
Step 3: Start chatting — that's it!
$ ollama run llama3.2
